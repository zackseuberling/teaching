Units for measuring in CSS

  • px - pixels
  • % - percentage
  • em - relative to the current element font-size
  • rem - relative to the root element font-size
  • vw - viewport width
  • vh - viewport height

Children + sibling selectors

            
              div {
                font-size: 30px;
                background-color: red;
              }

              /* child selector */
              div span {
                font-size: 20px;
              }

              /* sibling selector */
              div + div {
                background-color: blue;
              }
