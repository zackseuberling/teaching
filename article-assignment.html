Assignment 1

Part 1 - Semantic HTML

Choose an interview or article from The Creative Independent, one with multiple levels of hierarchy and images. Translate this article into HTML. Focus on hierarchy and using the most specific elements to organize the content.

Include:

  • Prelude and artist photo
  • Interviewer/author name
  • Date of publishing
  • Article text
  • “Some Things” portion

Resources

  • Semantic Elements
  • The Creative Independent

Part 2 - CSS

Add CSS to the article. Focus on improving clarity, organization, and hierarchy. Make sure that the style is distinctly different from the Creative Independent’s visual style.

Look through the MDN CSS Reference. There’s a lot and many will probably seem too daunting or confusing, but don’t let that deter you! Each reference has a tutorial for implementing the property and using it. Try to get used to navigating the MDN website and following tutorials.
